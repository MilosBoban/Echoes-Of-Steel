# WebAplicationEchoesOfSteel
# Web aplikacija za prodavnicu Echoes Of Steel.

Web aplikacija za prodaju dekorativnog oruzja i oklopa iz razlicitih istorijskih epoha,
uz edukativni sadrzaj o njihovom proijeklu i znacaju.

---

## 🚀 Funkcionalnosti

- Pregled proizvoda  
- Dodavanje u korpu  
- Pretraga proizvoda  
- Registracija i prijava korisnika  
- Adminska kontrola aplikacije

---

## 🛠️ Tehnologije

- Frontend: Angular
- Backend: Express.js, Node.js
- Baza podataka: MongoDB
